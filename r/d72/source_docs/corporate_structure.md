## 3. Control Structure

### Class A — Allison Hensgen (Veto + Majority Control)

- Veto rights on all major decisions across all entities
- Manager designation — full operational authority
- Named patent holder on all filings
- Majority economic interest
- Final say on: IP transfer/licensing/sale, entity dissolution/merger, new equity issuance, debt above $50K threshold, strategic pivots, addition/removal of members

### Class B — Joel (Strategic Advisory)

- Advisory/economic position at Holdco level **only if expressly approved by Allison in writing** — no automatic interest
- If granted: narrowly defined advisory/economic only; no blocking rights; no dilution of Allison's Class A control
- Must be consulted on: technical architecture, AI/data infrastructure, cross-entity Holdco matters
- No day-to-day operational authority over RootWell
- In RootWell: may serve in non-executive strategy or chair-type role with no operational veto

### Class C — Aniroop Vitta (Restricted Economic Interest)

- Economic interest only — suggest 5–15%, to be negotiated
- No management rights, no veto
- Scoped exclusively to RootWell & Star OpCo
- 4-year vesting schedule with 1-year cliff recommended
- Subject to buy-sell clause
- Must execute IP Assignment Agreement before contributing any work

## 7. Critical Risk Flags

| Risk | Mitigation |
|---|---|
| Allison built IP before entity formation | Retroactive IP Assignment Agreement — execute immediately or prior work is unprotected |
| Aniroop contributes code without assignment | Execute IP Assignment before any work begins — no exceptions |
| Medical advice liability | All disclaimers in OpCo; carry E&O insurance; Holdco insulated |
| Equal partnership deadlock | Eliminated — Allison's Class A veto is the tiebreaker |
| VC investment incompatible with LLC | Add Delaware C-Corp conversion clause to Operating Agreement now |
| Cross-entity contamination | Aniroop's restricted agreement explicitly blocks cross-entity rights |
| IP theft by departing founder | All IP vests in Holdco at formation; individual assignment is irrevocable |
