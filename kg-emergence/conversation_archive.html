6. User Asked: Can We Mathematically Prove Common Sources Wrong?

The user asked whether the system can mathematically allow classically rejected hypotheses to become accepted.

Assistant response:

We cannot guarantee discovery of truth.
We can guarantee epistemic admissibility.

Core theorem-like condition:

For every empirical hypothesis H,
if P(H) > 0
and observations are not all dependent copies
and likelihood ratios can grow without artificial cap,
then H can defeat an initially dominant consensus hypothesis.

Key rules:

- No empirical hypothesis gets prior 0 or 1.
- Official sources have no infinite authority.
- Conspiracy sources have no zero value.
- Source reliability is domain-local.
- Consensus can lose if posterior evidence warrants.
