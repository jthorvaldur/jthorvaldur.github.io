Graphlet recurrence matrix

View C of the experiment design. Each row is one tradition's extracted role-graph; each column is a named graphlet template from the library. A cell's colour and number are how strongly that tradition instantiates that template's canonical edge/role skeleton (coverage in [0,1]). The bottom row counts how many independent traditions clear the recurrence threshold — a motif that lights up across many traditions is a cross-tradition structure, not a local accident.
