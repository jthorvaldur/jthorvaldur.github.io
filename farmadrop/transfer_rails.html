Stripe launches; a chain coin scales. Stripe is the Phase-0 rail because people know and trust it — but it taxes every dollar (2.9% + $0.55). The first deploy coin is a transfer-efficiency coin (CPDR-style) for cheap peer transfers — not the asset-backed Honey Coin, which is not ready and comes later via the compadres asset-backed template. This page's only job: make the cost gap between the two rails legible.

Section 4

The phased plan

Bootstrap trust on Stripe, remove the card-rail cost with an efficient transfer coin, and only then — with counsel — reach for the asset-backed token. Each phase is a decision gate, not just a date.

Phase 0 → Phase 1 → Phase 2 · gated, not automatic

Phase 1 is the first real cost reduction — the only mechanism that genuinely removes Stripe's fee (contrast with the PMA "fees-are-a-tax" framing, which is a legal characterization, not a mechanism; map §3.4.4). Phase 2 stays blocked on regulatory classification.
